avsubtitles
===========

一个基于libass/FFmpeg的字幕解码工具库，支持内挂以及外挂字幕解码。


##### 常用问题

* 因为代码仓库中不包含其它已编译好的三方库，如果有需要请到 https://sourceforge.net/projects/avplayer/files/develop/avsubtitles/ 下载已经编译好的字幕开发包。
